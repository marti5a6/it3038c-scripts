# Installation Instructions

In order to use this script, the python package installer (PIP) needs to be up to date.

* In PowerShell, enter "python -m pip install --upgrade pip"
* In Linux Terminal, enter "python3 -m pip install --upgrade pip"

Then simply run the script, named "sysinfo.py".

* In PowerShell, enter "python sysinfo.py" while in the directory.
* In Linux Terminal, enter "python3 sysinfo.py" while in the directory.

Lastly, follow the directions on-screen to present the desired system information.

Enjoy!
